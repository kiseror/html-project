Instructions

In the last activity, you were only tasked to make one-sided dice. This time, you'll be creating 6 sided dice/full-sided dice. Please see attached image for reference.

  1. Set the border of the dice to be 200px width x 200px height.
  2. Set the size of the small circles to be 40px height and 40px width.
  3. Use the CSS property "border-radius" to soften the edges of the divs and to create the small circles.
  4. Use CSS box model, display, and position properties to position the different div's to their proper places.
*Attached is the HTML file. You may alter or edit the HTML file if you want as long as the desired output is achieved. Use internal styling in making the activity so that you will only have to submit one file. Good Luck and Enjoy the activity!!
