Rotate
The rotate() method rotates an element clockwise or counter-clockwise.

transform: rotate(_deg);

This div is rotated 10deg clockwise

Mouse over the element to see the effect!

This code is made by Group 4 :3

This div is rotated 180deg counterclockwise (-180deg)
mouse over the element to revert to its original rotation

Translate
The translate() method moves an element from its current position (according to the parameters given for the X-axis and the Y-axis).
This div element is moved 37 pixels to the left, and 24 pixels down from its current position.
This div element is moved 10 pixels to the right, and 10 pixels up from its current position.

This div element is moved 30% up

Scale
The scale() method increases or decreases the size of an element (according to the parameters given for the width and height).

Mouse over the elements to see their effects!

transform: scale(_, _);

This div element is two times of its original width, and four times of its original height.
This div element is half of its original height and width.

transform: scaleX(_);
Increases or decreases the width of an element.

This div element is three times of its original width
This div element is quarter of its original width

transform: scaleY(_);
Increases or decreases the height of an element.

This div element is three times of its original height.
This div element is 3/4 of its original height.
Skew
The skew() method skews an element along the X and Y-axis by the given angles.
transform: skewX(_deg);
Skews an element along the X-axis by the given angle.
transform: skewY(_deg);
Skews an element along the Y-axis by the given angle.
This div is skewed 30deg along the x-axis
This div is skewed -5deg along the y-axis
This div is skewed -45deg along the x-axis
This div is skewed 20deg along the y-axis

transform: skew(_deg, _deg);

This div is skewed 56deg along the x-axis and -14deg along the y-axis
transform: skew(56deg, -14deg)

Mouse over the elements to see their effects!

This div is skewed 56deg along the x-axis and -14deg along the y-axis
transform: skew(-43deg, 3deg)

Matrix
The matrix() method combines all the 2D transform methods into one.
transform: matrix(1.25, 0, 3, 2, 0, 0);
transform: matrix(0.5, -0.5, 1, 3, 4, 3);
transform: matrix(1, 0.2, 0.3, 4, 5, -6);

The parameters are as follow:
matrix(scaleX(),skewY(),skewX(),scaleY(),translateX(),translateY())
