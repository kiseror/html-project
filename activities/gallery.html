Instructions

  1. Make all the fonts Helvetica.
  2. Center the headings.
  3. Align the images horizontally.
  4. Use CSS selectors that when you hover an image, it will have a border like that of in the video instructions.
Video Instructions: https://www.loom.com/share/3137851d410248eea3620c5873890a77

In the last activity, you're not allowed to edit the HTML Document. This time, I'm allowing you to alter or edit the HTML Document as long as the desired output is achieved. Use INTERNAL CSS so that you will only have to attach 1 file.

Code

      padding: 5px;
      text-align: center;
    }

      border: 5px solid gray;
    }
